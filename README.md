# 🤖 AI Study + Doubt Solver Platform

📚 "Study Smart, Not Hard"

An AI-powered study assistant designed to help students learn more efficiently by providing instant doubt solving, smart note summarization, quiz generation, and personalized study planning.

 🚀 Features

- 📄 **Notes Summarization**
  - Convert long notes into short, easy-to-understand summaries
  - Helps in quick revision

- 🤖 **AI Doubt Solver**
  - Ask questions and get instant answers
  - Supports multiple subjects

- 📝 **Quiz Generator**
  - Automatically generates quizzes from study material
  - Helps in self-assessment

- 📅 **Study Planner**
  - Creates personalized study schedules
  - Improves productivity and time management

🛠️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **Database:** SQLite  
- **AI Integration:** OpenAI API  
- **Libraries:** Pandas, Requests  

⚙️ How It Works

1. User uploads notes or asks a question  
2. Backend processes the input  
3. OpenAI API generates response  
4. Output is displayed (summary / answer / quiz)
